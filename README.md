# LLM-based Online Comments Summarizer (Backend)

Group B backend baseline for v1 APIs.

## Quick Start

1. Install deps

```bash
npm install
```

2. Create env

```bash
cp .env.example .env
```

3. Start server

```bash
npm run dev
```

Server default: `http://localhost:3000`

## API

- `GET /api/v1/health`
- `POST /api/v1/analyze`

See: `docs/API_USAGE.md`
